# Detexturify

Detexturify removes the custom texture pack that's forced by Hypixel SkyBlock and will replace the items with their original item models in the future.

## Why?

The custom Hypixel SkyBlock texture pack is not cool enough to be forced!

## Will this get me banned?

No.

## Can I still use my own texture packs?

Yes! You can load them normally like you would on a vanilla instance.

## Credits

This project uses data which is sourced from Noamm9's item list! The said item list is available [here](https://api.noamm.org/resources/skyblock-items).
